[h1]Manipulation Authority Framework[/h1]

A modding framework that allows creators to build custom rules and restrictions for moving, dismantling, or destroying world objects.

[list]
[*] [b]Three-Phase Manipulation Pipeline[/b] - Hooks for validation, pre-action, and post-action stages to control every part of a world interaction.
[*] [b]Priority-Based Rule Registration[/b] - A system that lets multiple different mods add their own protection rules without creating conflicts.
[*] [b]Comprehensive Action Support[/b] - Provides global control over sledgehammer destruction, furniture relocation, and structure disassembly.
[*] [b]Sandbox Security Controls[/b] - Server admins can easily configure protection settings and security limits through the standard sandbox menu.
[*] [b]Automatic World Validation[/b] - Rules trigger instantly whenever a player attempts to modify the environment to ensure base security.
[/list]

This framework is essential for creators building roleplay, base protection, and world persistence systems for multiplayer servers.
